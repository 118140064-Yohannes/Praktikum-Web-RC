CSS

CSS singkatan dari (Cascading Style Sheet adalah salah satu bahasa pemrograman desain web (style sheet languages) yang mengontrol format tampilan sebuah halaman web yang ditulis dengan bahasa penanda (markup language). Biasanya CSS digunakan untuk mendesain sebuah halaman HTML dan XHTML, tetapi sekarang bahasa pemrograman CSS bisa diaplikasikan untuk segala dokumen XML, termasuk SVG dan XUL.

CSS bekerja dengan menggunakan dua buah elemen penting untuk pemformatan tampilan, diantaranya elemen selector dan deklarator. Dua buah elemen ini berfungsi sebagai penentu format tampilan dan lainnya menempatkan format tampilan. Sedangkan selector adalah sebuah perintah lanjutan dari deklarator dan berfungsi menempatkan format tampilan dari deklarator pada halaman web.
